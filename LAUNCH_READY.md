### **Quick Start (5 Minutes)**
```bash
# 1. Clone and setup
git clone [your-repo]
cd TWIN

# 2. Backend setup
cd backend
poetry install
poetry run uvicorn app.main:app --reload --port 8000

# 3. Frontend setup (new terminal)
cd frontend  
npm install
npm start

# 4. Open http://localhost:3000
# 5. Connect your Obsidian vault and Notion workspace
```
